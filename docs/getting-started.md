# Getting started

1. Include `dist/aurora.min.css` in your page.
2. Use utilities for layout/spacing/typography.
3. Compose components like `.btn`, `.card`, `.alert`.
4. Toggle dark mode with `data-theme="dark"` on `<html>` or `<body>`.

## Breakpoints
- `sm` ≥ 640px, `md` ≥ 768px, `lg` ≥ 1024px, `xl` ≥ 1280px.

## State variants
Use attribute-style variants on the same element:
- `hover:underline` → write as `[hover:underline]`
- `focus:ring` → `[focus:ring]`
- `active:opacity-75` → `[active:opacity-75]`
